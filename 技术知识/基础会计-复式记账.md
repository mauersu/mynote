# 会计发展与会计目标
会计目标主要服务于企业外部的会计信息使用者进行相关的经济决策

# 会计要素与会计等式
会计要素包括资产、负债、所有者权益、收入、费用和利润

资产=负债+所有者权益

收入-费用=利润

会计方法由账户设置、复式记账、会计凭证填制和审核、账薄登记、成本计算、财产清查和财务报告编织七种具体方法构成

# 账户设置
账户是根据会计科目设置的，具有一定结构形式，用以连续、系统、全面地记录交易或事项，反应会计要素增减变动及其结果，并为会计报告的编织提供数据资料的一种工具。

# 复式记账
复式记账是指对企业发生的任何一项交易或事项都以相等的金额在两个或两个以上相互联系的账户中进行平衡记录，借以反应会计要素具体内容增减变化的记账方法。

我国现行《企业会计准则》规定：“企业应当采用借贷记账法记账”，记账规则：有借必有贷，借贷必相等。
